## 🧭 What this app is

chub-supply-chain-poc is a Windows app for testing how AI docs and model tools can affect software projects through hidden prompts and injected content. It helps you inspect a controlled proof of concept that focuses on supply chain risk, prompt injection, and MCP server behavior.

This project is built for local use on a Windows PC. You can download it, open it, and run it from the release page without needing to write code.
